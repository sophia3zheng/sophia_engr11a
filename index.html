Nametag

My first project was a nametag where I downloaded from the project outline which did not include any documentation. My workflow steps started with importing the nametag stl file into PrusaSlicer and then editing the texts to print my name and aligning it to make it look better. Then I exported it onto a USB drive and filled out the job number request form and put down my information on a sticky note. I inserted the USB into the Prusa Mini printer and let it print. It turned out well however I realized only after I was done printing that the color change did not work. I relearned how to do a color print and I will attempt it on a future project. Something I learned is how to load filament into the printer for color switches to just start a print.

   

Torture Test Spider

I downloaded a spider torture test from printables that tests overhangs, cooling, bed adhesion, and retraction based on the documentation. The creator recommended if we want to test anything to not try to do multiple chances all at once. They also recommended maximum cooling and a brim if you don’t have great bed adhesion but since we are testing the limits of the printer, I didn’t add or change anything. After printing it, the results turned out really well despite not having any supports and rafts. The only thing that I didn't expect was that the PLA was very stringy because of the printer settings not sucking the filament back in when it switches locations. The spider also did not print the x symbol on its head most likely because I printed it on its head and the bed of the printer melted it smooth. It is around 1.7x2.1 inches which is accurate. Something I learned from this project is how versatile 3D printing is. There is so much you can print and you never know if you don't try printing it. You can really push the limits a lot but some limits include extreme overhangs (if you don’t include supports), accuracy (if the size is too small), etc.

   

Nintendo Game Controller (single)

I downloaded a single Nintendo game controller holder from printables to print as my open-source assistive device. It is made for anyone who likes playing Nintendo. Sometimes when you are playing multiplayer, you only get one side of the controller and this will make it easier to hold without getting cramps in your hand from how small the controller is. It was made by Semmel on Printables. There was little to no documentation mostly because the print is very straightforward so it didn’ need much documentation the documentation is pretty efficient. The documentation told me I didn’t need support, however I still added support just in case because the controller does not print flat on the ground. Some changes I would suggest would be that we would probably simplify the design because this print is very bulky and uses up a lot of filament. My process involved downloading and importing it into Prusa and printing it with organic support. After it was done printing, I removed the supports with pliers. It turned out well, I just kept removing pieces that were necessary. There are parts that are very delicate that I almost broke. Sometimes I learned that I have to be very careful when using pliers or I could potentially ruin the print.
